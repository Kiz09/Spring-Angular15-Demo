
# SpringAngularDemo

## Execute

Inside main folder
- Run 'docker compose up'

### Development server Spring

Inside root folder
   - Run `mvn spring-boot:run`
   - Navigate to `http://localhost:8080/payments/`

### Development server Angular

Inside root folder 
- Run: `npm i`
- Run: `ng serve --port 8081`
- Navigate to `http://localhost:8081/payments`.
